URL Shortener Microservice API

Coded by Joshua Davidson

Description

A simple url shortener microservice API that returns a shortened URL within a JSON or redirects if passed a shortened URL.

Pass a URL to shorten or a shortened URL to visit to https://url-microservice-project.herokuapp.com/ to give it a try!

Example Usage

https://url-microservice-project.herokuapp.com/https://www.freecodecamp.com

Example Output

{ originalURL: "https://www.freecodecamp.com", shortURL: "https://url-microservice-project.herokuapp.com/free" }
